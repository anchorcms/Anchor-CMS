Basic Configuration

Fine turning Anchor.

Database

Your database settings are found in the config.php there are required to run Anchor:

'database' => array(
	'host' => 'localhost',
	'username' => 'myusername',
	'password' => 'mypa33wr0d',
	'name' => 'anchorcms'
)
For most hosting the database host will be localhost.

Metadata

By default you will have these setting:

'metadata' => array(
	'sitename' => 'My Site Name',
	'description' => '',
	'typekit' => '',
	'twitter' => '',
	'date_format' => 'jS M, Y'
)

Change these to suit your needs.
