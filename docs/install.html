Requirements & Installation

Getting things up and running in Anchor will be a breeze.

Requirements

When deploying Anchor will need installed on your server the following:

  • Apache web server.
  • PHP 5.3
  • MySQL 5

Installation

A few basic steps to follow:

  1. Download Anchor
  2. Extract the archive and upload the contents to your web server.
  3. Navigate to your application in a web browser and follow the install guide.
