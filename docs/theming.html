Theming Anchor

Theming Anchor CMS aims to be easy.

Files

A theme for Anchor requires a few basic files:

  • about.txt
  • index.php
  • article.php
  • page.php
  • posts.php
  • search.php

There are a number of optional files you can also include:

  • includes/header.php
  • includes/footer.php
  • 404.php

Functions

Inside each theme template you can use a php function to retrieve dynamic data. Here is a list of all the available defined functions:

posts()
Lists all avaiable posts.
